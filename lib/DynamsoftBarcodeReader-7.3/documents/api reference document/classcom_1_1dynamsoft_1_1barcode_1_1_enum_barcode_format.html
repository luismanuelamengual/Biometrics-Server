Detailed Description

Describes the type of the barcode. All the formats can be combined; such as BF_CODE_39 | BF_CODE_128. Note: The barcode format our library will search for is composed of BarcodeFormat group 1 and BarcodeFormat group 2, so you need to specify the barcode format in group 1 and group 2 individually.
