Sets whether or not to use percentages to measure the Region size.

Value range:
[0, 1]
Default value:
0
Remarks:
When itos set to 1, the values of Top, Left, Right, Bottom indicates the percentage (from 0 to 100). Otherwise, they refer to the coordinates. 0: not by percentage 1: by percentage
